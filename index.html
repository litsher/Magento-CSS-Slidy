CSS Slidy Magento Module

Magento pure CSS3 Slider (Magento 1)

View on GitHub Download .zip Download .tar.gz

What is it

Pure CSS3 Slider for Magento. The simplest and fastest and most hassle-free thing ever.

How to use it

Copy the module files into your file system.
Upload images of equal size into media/cssslidy/slider
Display CSS Slider on Magento CMS pages: {{block type="CSSSlidy_Slider/render"}}

More info in README
Tested on Magento 1.9
